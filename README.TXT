PROJECT TITLE: WordsCounter
PURPOSE OF PROJECT: implementing words counters in given files, in single and multiple mode, both sequentially and multithreaded
VERSION: 1.0
HOW TO START THIS PROJECT: there are few ways to start this project, depends on the mode selected. the main method in WordsCounter runs a single wordCounter, the main method in MultiWordsCounter runs a triple wordCounters sequentially, the main method in ParallelMultiWordsCounter runs a triple wordCounters in parallel
